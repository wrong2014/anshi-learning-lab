# 三科 × 原因因子诊断矩阵 v01

整理日期：2026-06-26

## 使用说明

本矩阵是内部判断框架，不直接展示给用户。前台只能展示“学习卡点定位结果”和行动建议，不展示完整因子权重与判断路径。

## 共通因子

| 编号 | 因子 | 核心判断问题 | 家长对话探针 | 孩子对话探针 | 适合的 UI |
|---|---|---|---|---|---|
| F01 | 前置知识缺口 | 旧知识是否调用不出来 | 最近卡住时，是否牵涉上一个单元或上学期内容？ | 这题里有没有哪个旧知识你记不准？ | 多选 chips |
| F02 | 概念理解不稳 | 是否只会背，不会解释 | 他能不能不用公式讲出这个概念是什么意思？ | 如果不用课本话，你会怎么解释它？ | 短文本 |
| F03 | 学科语言与符号理解困难 | 题干、术语、符号是否读不准 | 他是不是经常读错题目里的关键词？ | 题目里哪句话或哪个符号最让你不确定？ | 题干标注 |
| F04 | 表征转换困难 | 文字、图、式子是否转不过来 | 他能不能把题目画成图或列成关系？ | 你能把这句话画出来或写成式子吗？ | 选择 + 画图上传 |
| F05 | 建模与迁移困难 | 是否不会建立变量关系 | 例题会，换个情境是否就不会启动？ | 你不知道选哪个公式，还是不知道有哪些量相关？ | 场景卡 |
| F06 | 程序执行不稳定 | 步骤、计算、单位、检查是否失控 | 错误更像不会，还是会但总漏步骤？ | 你最后检查时通常检查什么？ | 多选 chips |
| F07 | 元认知与复盘薄弱 | 是否说不清自己哪里不会 | 他能不能说出从哪一步开始不会？ | 你能指出第一处卡住的位置吗？ | 断点选择 |
| F08 | 学习策略低效 | 是否缺少有效练习闭环 | 错题之后通常做什么？ | 看完答案后你会隔天再做一次吗？ | 流程选择 |
| F09 | 情绪动机与自我效能受损 | 是否怕错、回避、自我否定 | 他遇到难题第一反应是什么？ | 看到这类题时你更像紧张、烦、还是不知道从哪开始？ | 程度滑条 |
| F10 | 家庭支持与 AI 使用失位 | 父母或 AI 是否替代思考 | 你通常是讲完整解法、催促、还是让 AI 讲？ | 你问 AI 时，是要答案还是让它追问你？ | 多选 chips |

## 数学特异信号

| 因子 | 数学中常见表现 | 区分性追问 |
|---|---|---|
| F01 | 分数、方程、函数、几何基础调用不稳 | 卡住时是否先卡在旧知识？ |
| F02 | 定义会背，但不会举反例或解释本质 | 能不能用自己的话解释这个概念？ |
| F03 | 符号、条件、关键词读不准 | 题干里哪个条件被漏掉或误读？ |
| F04 | 文字题不会转式子，几何条件不会转关系 | 能不能把一句话变成一个式子或图上标记？ |
| F05 | 例题会，变式不会；函数、几何综合题不会启动 | 你是在选方法时卡住，还是在第一步就卡住？ |
| F06 | 步骤跳跃、符号错、计算错、证明漏逻辑 | 错误通常出现在第几步？ |

## 物理特异信号

| 因子 | 物理中常见表现 | 区分性追问 |
|---|---|---|
| F01 | 力、运动、压强、电路等基础概念断层 | 这题需要的旧概念是否能说清？ |
| F02 | 会背公式，不理解量的意义 | 每个物理量代表什么能说清吗？ |
| F03 | 题干现象、单位、方向、图像读不准 | 题目中哪个条件决定了方向或过程？ |
| F04 | 不会画受力图、过程图、运动图像 | 如果先画图，能画出过程吗？ |
| F05 | 不知道用牛顿、能量、压强、电路哪个模型 | 你是在选公式时卡住，还是不知道过程怎么变？ |
| F06 | 单位、正负号、方向、代入计算混乱 | 错误是公式选错，还是代入和单位出错？ |

## 化学特异信号

| 因子 | 化学中常见表现 | 区分性追问 |
|---|---|---|
| F01 | 元素、化合价、离子、酸碱盐基础不稳 | 这题需要的基础表或规律是否熟？ |
| F02 | 会背反应规律，不理解为什么发生 | 这个反应为什么能发生能说清吗？ |
| F03 | 术语、符号、化学式、方程式读写不稳 | 哪个符号或名称最容易混？ |
| F04 | 宏观现象、微观粒子、符号方程式对不上 | 现象、粒子变化、方程式能否互相对应？ |
| F05 | 反应规律不会迁移到新物质 | 新物质出现时是否不知道套哪条规律？ |
| F06 | 配平、计算、实验题步骤失控 | 错误在配平、计算、还是实验信息提取？ |

## 内部隐藏策略

1. 不直接问“是不是 F05 建模困难”，而问最近一次具体卡点。
2. 同一个因子至少需要两类证据支持，才作为主因。
3. F09 情绪动机类只作为学习支持风险信号，不做心理诊断。
4. F10 家庭支持类必须谨慎表达，避免让父母感到被指责。
5. 若 F09 和 F10 同时高，建议人工初筛，不直接给强结论。
