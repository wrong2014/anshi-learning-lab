# 产品总架构_当前有效版 v01

整理日期：2026-06-26

## 当前产品路径

```text
内容 / 直播 / 私信
  -> P01 学习卡点定位工具
  -> P02 家长 AI 学习支持实操课
  -> P03 一户一案家庭学习支持方案
  -> 按需工具交付 / 专属轻量微应用
```

## P01 学习卡点定位工具

角色：免费或低门槛入口。

核心交付：

- 当前最优先的支持方向。
- 父母最容易帮错的地方。
- 接下来 7 天先停止什么、先做什么。
- 推荐进入 P02 或 P03 的路径。

分类首版：

1. 理解没有长出来。
2. 题目转化和迁移卡住。
3. 执行与复盘不稳定。
4. 家庭支持与 AI 使用失位。

## P02 家长 AI 学习支持实操课

角色：一对多产品，用于建立信任、验证需求、产生基础收入。

推荐形式：

- 2 次直播实操课。
- 7 天家庭练习。
- 模板包。

只讲四个场景：

1. 孩子说“我不会”。
2. 一道错题总错。
3. 父母没有时间陪。
4. 孩子向 AI 要答案。

## P03 一户一案家庭学习支持方案

角色：唯一对外主推的一对一核心服务。

服务周期：第一阶段固定 30 天。

核心承诺：

> 不替父母补数理化；帮助父母重新看清问题、做对支持，并跑出第一个能持续的家庭学习闭环。

服务模块：

| 阶段 | 内容 |
|---|---|
| 证据收集 | 试卷、错题、家长观察、学习节奏、当前资源 |
| 支持判断 | 判断当前最优先问题与家庭误区 |
| 家庭方案 | 明确父母、孩子、AI、外部资源分工 |
| 父母指导 | 教父母提问、停手、记录、使用 AI |
| 工具落地 | 配置必要工具，必要时共创小工具 |
| 复盘调整 | 根据执行反馈修改方案 |

## 工具交付原则

工具是 P03 的可选模块，不是独立主产品。只有当同一个问题跨多个家庭重复出现，并且人工支持方式已经验证有效时，才考虑模板化、工具配置、共创或定制微应用。

## 产品验收标准

- 父母能否更清楚地描述孩子卡在哪里。
- 父母是否减少无效讲题、催促、乱换资源。
- AI 是否从答案机变为追问、复测、记录工具。
- 家庭是否能稳定完成一个小闭环。
