# 课标驱动知识图谱生产链路

## 三个内部生产阶段

课程大纲、原子知识点和知识关系是三种不同问题：大纲解决“范围是否完整”，知识点解决“叶子里有什么”，关系解决“先学什么、如何迁移”。系统固定采用：

```text
教育部官方 PDF
  -> SHA-256 核验与逐页 OCR
  -> 确定性课标种子（模型不可删改）
  -> 大纲阶段（多模型候选 -> 硬校验 -> Supervisor）
  -> 知识点阶段（每次只处理一个叶子，多模型候选 -> Supervisor）
  -> 覆盖账本（65/65 才能继续）
  -> 关系阶段（多模型候选 -> 图校验 -> Supervisor）
  -> 引用、覆盖与图结构硬校验
  -> CurriculumKnowledgeNetwork
```

关系阶段按课标主题分批建立内部关系，最后单独建立跨主题、跨学段和跨学科关系，避免把完整九年知识网络塞进一次模型调用。

数学、物理、化学是三个领域智能体；大纲、知识点、关系是领域智能体内部阶段。Supervisor 是每个阶段的质量闸门。

## 官方来源

统一入口为教育部《[关于印发义务教育课程方案和课程标准（2022年版）的通知](https://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html)》。只使用该通知链接的数学、科学、物理和化学 PDF；下载哈希记录在 `data/official_curriculum_sources.json`。

## 任务树边界

- 数学：1-9 年级；四个学习领域；按 1-2、3-4、5-6、7-9 年级拆分。
- 科学：1-6 年级；课标目录列出的 13 个学科核心概念；按三个小学学段拆分。
- 物理：8-9 年级；物质、运动和相互作用、能量、实验探究、跨学科实践。
- 化学：9 年级；科学探究与化学实验、物质的性质与应用、物质的组成与结构、物质的化学变化、化学与社会·跨学科实践。

种子规定权威骨架，不声称它本身已经包含全部原子知识点。

## 覆盖与引用纪律

每个 `stage_task` 有 `expected_min_points`。知识点阶段只接收该任务引用页范围内的 OCR 文本，输出的每条引用必须满足：

1. 来源属于该任务；
2. 精确到一个课标逻辑页；
3. 摘录能在该页 OCR 文本中找到；
4. 年级和学科不越过任务边界。

覆盖账本由代码重新计算，模型不能自行宣布“已覆盖”。

## 小学科学承接

小学没有独立物理、化学课程。当网络同时包含科学、物理、化学时，必须分别存在科学到物理、科学到化学的 `bridges_to`。实际生产重点覆盖物质结构、物质变化、运动与相互作用、能量、科学探究与工程设计五组承接关系。
