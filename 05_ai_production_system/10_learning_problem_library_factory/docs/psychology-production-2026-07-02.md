# 第三层心理与认知真实生产记录（2026-07-02）

## 结论

青少年学习心理与认知发展资料已完成真实双候选生产、Supervisor 审查、本地严格验证、跨批次关系闭包和全局审计。

当前状态为 `human_review_required`，不是可直接面向用户发布的心理评估或干预工具。正式发布必须同时满足：教育心理/临床专业人员逐条核验来源与内容，以及具名人类审批者签署。

- 完美主义相关困扰；
- 考试焦虑；
- 学校倦怠。

认知层 4 个：

- 认知负荷；
- 工作记忆；
- 注意调控；
- 元认知监控。

动机层 3 个：

- 内在与外在动机；
- 成就目标取向；
- 自我决定与基本心理需要。

## 来源与边界

- 理论来源：12 份经典理论或原始研究证据卡；
- 安全来源：WHO 青少年心理健康、UNICEF 青少年求助指南、NIMH 儿童青少年预警与求助指南；
- 每个维度均同时引用对应 `theory-*` 来源和至少一个 `guideline-*` 来源；
- 来源卡是对原始论文/指南的保守摘要，`verified_by_human=false`；
- 习得性无助采用 Maier 与 Seligman 2016 年修订，不把早期动物实验直接外推为学生诊断；
- “完美主义相关困扰”明确为观察性教育构念，不是诊断名称；
- 美国来源中的具体热线号码不直接移植到中国场景，产品只提示联系所在地监护人、学校支持、医疗机构、心理援助热线或紧急服务。

## 生产中发现并修正的问题

1. 第一轮认知层通过，但动机层某执行器连续六轮写入禁用主观标签，因无法形成双候选而停在 `needs_human`。
2. 提示词强化后仍会在引语或复合词中偶发禁词，因此增加确定性中性化：在进入验证和监督前改写为可观察的中性表述，并记录具体字段路径 warning。
3. 第二轮 Supervisor 要求 Schema 不存在的 `teacher_signals`，协议已明确心理资料只有家长信号、学习者信号和数理化场景，不允许因虚构字段扣分。
4. 第二轮还出现分项加权应为 92、报告写成 93 的算术错误；总分现由代码按量表权重计算，模型只负责分项评价和语义问题。
5. 第三轮成功完成。最终选中候选只有一次中性化 warning，位于动机批次的化学场景描述，路径已写入工作包。
6. 原始跨批次关系有 36 处单边视图；工作包按 `A may_lead_to B` 等价于 `B may_be_caused_by A` 做确定性闭包，得到 42 条关联和 0 个视图不一致。所有关系仅表示待验证关联假设，不是临床因果结论。
