# 内容转化路径 v01

整理日期：2026-06-26

## 内容任务

内容不是展示 AI 多厉害，而是让父母看见自己的真实困境，并知道下一步可以做什么。

## 标准路径

```text
短视频 / 直播 / 长文
  -> 指出一个父母真实困境
  -> 给一个今晚能做的小动作
  -> 引导做 P01 定位工具
  -> 根据结果进入 P02 或 P03
```

## 内容主题池首版

| 主题 | 钩子 | 落点 |
|---|---|---|
| 孩子说我不会 | 别急着讲题，先让孩子说清从哪一步不会 | P01 |
| 补课没效果 | 可能不是老师不行，而是问题没被看见 | P03 |
| AI 变答案机 | 真正该管的不是 AI，而是家庭使用规则 | P02 |
| 父母不会讲题 | 不懂数理化，也能做过程支持 | P02 |
| 错题反复错 | 错题本不等于复盘，48 小时复测更关键 | P01 |

## 每条内容的最低要求

- 一个真实家庭场景。
- 一个父母容易误解的点。
- 一个可执行动作。
- 一个明确边界。
- 一个自然转化入口。

## 禁止表达

- 不制造父母焦虑。
- 不攻击孩子。
- 不把 AI 包装成万能。
- 不用空泛教育话术替代具体动作。
